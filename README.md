# BEND FORGE

板金×設計 ── 朝15分・触って曲げて覚える3D道場（スマホPWA）

- 遊ぶ: https://pasarinobuhito0409-droid.github.io/bend-forge/
- 型: 予想 → 3D実験（Three.js）→ 種明かし（説明カード1枚＝専用3Dアニメ）→ ミニクイズ → スタンプ
- 第1章「板金を見る」全6回＋ボス戦収録（板金とは・曲げ・展開・スプリングバック・曲げ近くの穴・バーリング）
- 🔬ミクロの目：原子の滑り（塑性）とバネ成分（弾性）でスプリングバックの正体まで見える
- 第2章「図面と現場へ」（曲げ順序・公差・溶接組立）は次のアップデートで配信予定
- 兄弟アプリ:
  - ARC FORGE（溶接） https://pasarinobuhito0409-droid.github.io/arc-forge/
  - FORCE FORGE（力学×CAE） https://pasarinobuhito0409-droid.github.io/force-forge/
  - VOLT FORGE（電気） https://pasarinobuhito0409-droid.github.io/volt-forge/
